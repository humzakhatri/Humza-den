The old site was a Bootstrap/jQuery template I'd had sitting there for years. Never touched it, never published anything on it. It was one of those things that just sat there being slightly embarrassing.

Scrapped it today.

New stack is Astro — Markdown files, GitHub Pages, GitHub Actions for deploy. Completely free, no CMS, no nonsense. Want to write something? Create a `.md` file, push, done.

The idea is the DEN — Daily Engineering Notebook. Not a blog, not a portfolio. Just a logbook of what I'm building and learning. I write, I push, I move on.

Kept the design very minimal. Pastel colours, clean fonts, a `/now` page as the main entry point with an archive on the side. Nothing fancy.

Rough is fine. It's live. That's enough for today.
